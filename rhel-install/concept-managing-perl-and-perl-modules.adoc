= 管理Perl和Perl模組
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
支援Perl命令的工作流程作業。OnCommand Workflow Automation您可以安裝及設定偏好的Perl發佈與Perl模組。

安裝WFA時、也會安裝NetApp Manageability SDK所需的Perl模組。成功執行Perl命令需要NetApp Manageability SDK Perl模組。

如果需要、您可以從Red Hat套件儲存庫或CPAN儲存庫安裝額外的Perl模組。
