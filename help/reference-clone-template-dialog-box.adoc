= 複製範本對話方塊
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
「Clone Template（複製範本）」對話方塊可讓您建立範本複本並編輯範本。

* *名稱*
+
可讓您編輯範本名稱。

* *類型*
+
可讓您從清單中選取一個字典物件。您選取要複製的範本名稱會做為複本名稱、並依預設附加複本。

* *實體版本*
+
可讓您以「major.mine.re版本」格式輸入範本的版本編號、例如1.1.0。

* *說明*
+
可讓您編輯範本的說明。

* *屬性*
+
顯示所選字典物件的屬性、並可讓您輸入每個屬性的值、例如enum和Function。





== 命令按鈕

* *儲存*
+
儲存範本並關閉對話方塊。

* *取消*
+
不會儲存範本並關閉對話方塊。
