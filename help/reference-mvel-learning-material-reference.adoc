= 何處可找到有關VEL的資訊
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用_MVEL Language Guide_來深入瞭解MVFlex運算式語言（MVEL）。

VEL是以Java語法為基礎的運算式語言。您可以在OnCommand Workflow Automation 諸如函數和變數等方面的動作流程中使用VEL運算式語法。
