設定WFA時、您可以視需要設定下列一項或多項：

* 將不支援的訊息傳送給技術支援部門AutoSupport AutoSupport
* Microsoft Active Directory輕量型目錄存取傳輸協定（LDAP）伺服器、適用於WFA使用者的LDAP驗證與授權

* 用於遠端資料記錄的系統記錄
